Titanic Survival Analysis using Python
📌 Project Overview
This project was completed as part of Task 2 of the Data Science with Python Internship.

The objective is to perform Exploratory Data Analysis (EDA) on the famous Titanic Dataset to understand how various passenger characteristics influenced survival during the Titanic disaster. The project demonstrates the complete data analysis workflow, including data cleaning, statistical analysis, and data visualization using Python.

🎯 Objectives
Load and explore the Titanic dataset.
Clean and preprocess missing data.
Analyze survival patterns based on passenger characteristics.
Create meaningful visualizations to support the analysis.
Draw insights from the dataset through exploratory data analysis.
🛠 Technologies Used
Python
Pandas
Matplotlib
Seaborn
📂 Dataset
The project uses the Titanic Dataset, one of the most widely used datasets for learning data analysis and machine learning.

The dataset contains information such as:

Passenger Class (Pclass)
Passenger Name
Gender (Sex)
Age
Fare
Embarked Port
Survival Status
🧹 Data Preprocessing
Before performing the analysis, the dataset was cleaned by:

Inspecting the dataset structure.
Identifying missing values.
Filling missing values in the Age column using the median.
Filling missing values in the Embarked column using the mode.
Removing the Cabin column due to excessive missing values.
📊 Analysis Performed
1️⃣ Survival Rate by Gender
Analyzed the survival rates of male and female passengers.

Observation

Female passengers had a significantly higher survival rate than male passengers.
2️⃣ Survival Rate by Passenger Class
Examined how passenger class influenced survival.

Observation

First-class passengers had the highest survival rate.
Third-class passengers had the lowest survival rate.
3️⃣ Distribution of Passenger Ages
Visualized the age distribution of all passengers using a histogram.

Observation

Most passengers were young adults.
The dataset includes passengers ranging from children to senior citizens.

💡 Key Insights
Female passengers were significantly more likely to survive than male passengers.
Passenger class had a strong impact on survival probability.
First-class passengers experienced the highest survival rates, while third-class passengers had the lowest.
Most passengers belonged to the young adult age group.
Proper data preprocessing is essential for obtaining reliable analytical insights.
